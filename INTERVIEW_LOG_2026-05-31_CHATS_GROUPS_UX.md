# Интервью — Чаты, группы, UX polish

> Дата: 2026-05-31
> Роль: product designer
> Контекст: список из 20 задач (баги + фичи) для collab_notes v1.17+

---

## Решения по пунктам

### A. Чаты и сообщения

| # | Задача | Решение |
|---|--------|---------|
| A1 | Кракозябры в инпуте сообщений | Баг — encoding UTF-8. Починить. |
| A2 | Кракозябры в bottomSheet отправки фото | Тот же баг encoding. Починить. |
| A3 | Сообщения не загружаются / не обновляются | Баг — нужно расследование. |
| A4 | 400 ошибка при вставке изображения с десктопа | Баг. |
| A5 | Статусы сообщений | **2 статуса**: 1 галочка = отправлено, 2 галочки = прочитано. Без "доставлено". В групповых: прочитано = хотя бы 1 участник прочитал. |
| A6 | Индикатор "печатает..." | Расположение: **над инпутом** (полоска над полем ввода). Формат: "Имя печатает..." / "Имя, Имя печатают..." |
| A7 | Аватарка в хедере личного чата | **Аватар + dot-индикатор + subtitle** с временем онлайна ("В сети" / "Был(а) 5 мин назад"). |

### B. Группы

| # | Задача | Решение |
|---|--------|---------|
| B1 | Убрать кол-во участников | Заменить на **превью последнего сообщения** (аватар + название + превью сообщения). |
| B2 | Контекстное меню группы (hold) | 5 пунктов: Список участников, Изменить название, Изменить аватарку, Пригласить, + **Удалить группу (только owner) / Покинуть группу (остальные)**. |
| B3 | Редизайн настроек группы | Заменить BottomSheet на **полноэкранную страницу** (как Telegram Info). |
| B4 | Pending-приглашения | Перенести из настроек в **окно просмотра группы** (рядом со списком участников). |
| B5 | Страница пользователя | Аватарка, история аватарок, username, имя, общие группы + **статус онлайн/время** + **кнопки: "Написать сообщение", "Пригласить в группу"**. Hold 300ms по участнику → контекстное меню (переход на профиль + исключение для owner/admin). |

### C. Уведомления

| # | Задача | Решение |
|---|--------|---------|
| C1 | Пуши чеклистов/заметок | **Отключить по умолчанию**, но дать возможность включить в настройках приложения. Настраиваемо. |
| C2 | Пуш новых версий | **Пуш со ссылкой** на скачивание APK/EXE. |

### D. Онлайн-статус

| # | Задача | Решение |
|---|--------|---------|
| D1 | Dot-индикаторы + "был в сети" | **Везде**: список чатов, список участников группы, профиль, хедер чата. Таймаут: **3 минуты** бездействия → offline. |

### E. Поиск пользователей

| # | Задача | Решение |
|---|--------|---------|
| E1 | Контакты при поиске | **Перед вводом** — когда инпут пуст, показывать список контактов (с кем был контакт) под заголовком "Недавние". |
| E2 | BottomSheet заползает под toolbar | Баг. Починить отступ от клавиатуры. |

### F. UI polish

| # | Задача | Решение |
|---|--------|---------|
| F1 | Анимация заметок | **Layout animation** — плавный морфинг карточек между grid и list (Hero-подобная анимация). |
| F2 | Toast — цвет кнопки | Текст кнопки сделать **белым**. |
| F3 | Фильтры Активности | **Chips под хедером** (горизонтальный скролл): "Все", имена пользователей, названия групп. |
| F4 | Скелетоны | **Shimmer** (мерцающие плашки) для всех загружаемых элементов на всех страницах. |
