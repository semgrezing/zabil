# Changelog

## 1.9.0+10 — 2026-05-29

### Деплой
- Backend: Docker rebuild + prisma migrate deploy на achiemvemer.ru
- APK: `app-release.apk` (55.6 MB) загружен на сервер
- Исправлены `catch((e) =>` → `catch((e: unknown) =>` в 5 файлах (strict TS в Docker)
- Исправлен JVM OOM при Gradle build (`-Xmx2G -XX:+UseSerialGC`)

### Activity feed (P2)
- Новый backend-модуль `/activity/feed` (GET) — виртуальный агрегированный фид из существующих таблиц (заметки, сообщения, участники) без новой миграции
- Flutter: `ActivityItem` модель, `ActivityService`, `ActivityFeedNotifier`, `ActivityFeedScreen`
- Доступ: Настройки → «Активность»; навигация кликом на item (заметка / групп. чат)
- Маршрут `/activity` добавлен в GoRouter

### Закреплённые заметки (P1)
- Поле `pinned` в Prisma-схеме (`Boolean @default(false)`) + индекс
- SQL-миграция `20260529000000_add_note_pinned`
- Сортировка: закреплённые заметки идут первыми (`orderBy: [{ pinned: desc }, { updatedAt: desc }]`)
- Zod: `pinned: z.boolean().optional()` в `createNoteSchema` и `updateNoteSchema`
- Flutter: поле `pinned` в `NoteModel`, `togglePin()` в `NotesNotifier`, `pinned` в `NotesService.updateNote`
- UI: иконка булавки на карточке + пункт «Закрепить/Открепить» в popup-меню

## 1.7.0+8 — 2026-05-29

### Навигация
- Сокращена нижняя навигация с 5 до 3 вкладок: **Заметки | Чаты | Настройки**
- Вкладка «Поиск» (поиск пользователей) перенесена в Настройки → «Найти пользователя»
- Вкладка «Приглашения» перенесена в Настройки → «Приглашения» (с бейджем непрочитанных)
- Увеличен гэп между иконками и подписями в навбаре (8 px)

### Редактор заметок
- Авто-сохранение с debounce 1.5 с (удалена ручная кнопка «Сохранить»)
- Индикатор статуса в AppBar: «Сохранение...» / «Сохранено» / «Заметка»
- Drag-and-drop перестановка пунктов чеклиста (`ReorderableListView`)
- Сохранение при выходе, если есть несохранённые изменения

### Карточка заметки (визуальное обновление)
- Цветная полоска слева (4 px) + тонированный фон по цветовой метке
- Прогресс-бар чеклиста (N из M + LinearProgressIndicator)
- Строка «Обновлено [имя], [когда]» внизу карточки
- Swipe-действия: влево — архивировать, вправо — удалить (с подтверждением)
- Чип группы над заголовком, превью изображений
- Переработана типографика и отступы на основе дизайн-токенов

### Исправления багов
- **LateInitializationError** — `late final _service` заменён на getter `get _service` в 7 нотифаерах (auth, groups, chats ×3, notes, invitations)
- **DioException 400 при установке цветовой метки** — каскадная проблема из-за LateInit; исправлена заменой паттерна
